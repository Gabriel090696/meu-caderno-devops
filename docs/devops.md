# # DevOps — Resumo 

Este material foi elaborado com o objetivo de consolidar e reforçar o aprendizado dos principais conceitos abordados na disciplina de DevOps. A proposta é organizar de forma clara e estruturada os conteúdos estudados, facilitando a revisão e contribuindo para a fixação do conhecimento.

## O que é DevOps
DevOps é a integração entre desenvolvimento (Dev) e operações (Ops) com o objetivo de entregar software de forma mais rápida, segura e com maior qualidade.

## Objetivo
- Melhorar a colaboração entre equipes 
- Automatizar processos  
- Aumentar a velocidade de entrega  
- Garantir qualidade e estabilidade  

## Ciclo DevOps
1. Planejar (Plan)  
2. Desenvolver (Code)  
3. Compilar (Build)  
4. Testar (Test)  
5. Liberar (Release)  
6. Implantar (Deploy)  
7. Operar (Operate)  
8. Monitorar (Monitor)  

Ciclo contínuo baseado em melhoria constante.

## Conceitos importantes
- CI (Integração Contínua): testes automáticos a cada alteração  
- CD (Entrega Contínua): código pronto para produção  
- IaC (Infraestrutura como Código): gerenciamento via código
