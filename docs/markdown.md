## Markdown
Markdown é uma linguagem de marcação leve usada para escrever documentação de forma simples e organizada.

### Para que serve
- Criar arquivos `.md` (README, documentação)  
- Organizar anotações  
- Documentar projetos no GitHub  

### Sintaxe básica

#### Títulos
```md
# Título 1
## Título 2
### Título 3
